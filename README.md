# We Dont Need No Sticken Points

WDNNSP is a personal flight-redemption decision-support app for comparing credit
card points, airline miles, and cash flight options.

## Development

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Validation

```bash
npm run build
```

Tests are not configured yet.

## Scope

Current scaffold routes:

- `/dashboard`
- `/wallet`
- `/search`
- `/results`
- `/settings`

Live flight APIs, Firebase, booking logic, alerts, and account syncing are not
implemented in this scaffold.
